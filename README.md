# Ideate-and-implement-a-system-to-enhance-the-quality-of-education-in-rural-areas.
Offline education system for rural areas using ESP32/ESP8266. Acts as a Wi-Fi hotspot and captive portal to serve study materials, quizzes, and career guidance without internet. Stores content on SD/flash and logs student progress offline. Supports SDG4 &amp; SDG8 with a low-cost, processor-based solution.
